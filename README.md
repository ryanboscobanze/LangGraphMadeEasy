
# LangGraph Made Easy

This repository contains the companion notebooks for the **LangGraph Made Easy** course.

The goal of this course is to provide a clear, practical, and step-by-step introduction to building agentic systems using LangGraph, with a strong focus on concepts, patterns, and hands-on examples.

## Create and activate a virtual environment

```bash
python -m venv .venv
# macOS / Linux
source .venv/bin/activate
# Windows (PowerShell)
.venv\Scripts\Activate.ps1
```

---

## Install dependencies

```bash
pip install -r requirements.txt
```

---

## Launch Jupyter


"jupyter lab" or "jupyter notebook"

Open the notebooks in order for the best learning experience.

## Notes

Some notebooks may require API keys (e.g., for LLM access).
Store secrets locally using environment variables or a .env file.
Do not commit API keys or credentials to this repository.
